# 🚀 快速启动指南

## 5分钟快速开始

### Step 1: 安装依赖（1分钟）

```bash
# 进入项目目录
cd campus_secondhand

# 安装必需的包
pip install ttkbootstrap matplotlib numpy

# 或使用国内镜像加速
pip install ttkbootstrap matplotlib numpy -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Step 2: 启动服务器（30秒）

**Windows:**
```bash
python start_server.py
```

**Linux/Mac:**
```bash
python3 start_server.py
```

看到以下输出表示成功：
```
服务器启动成功，监听 localhost:8888
默认管理员账户创建成功
```

### Step 3: 启动客户端（30秒）

**打开新的终端窗口**，然后运行：

```bash
python start_client.py
```

### Step 4: 登录系统（1分钟）

**方式1：使用管理员账户**
```
用户名: admin
密码: admin123
```

**方式2：注册新账户**
- 点击"注册新账号"
- 填写信息并注册
- 返回登录

### Step 5: 生成测试数据（2分钟）

1. 使用管理员登录
2. 点击"🎲 生成测试数据"
3. 设置数量：
   - 用户数量: 10
   - 商品数量: 30
   - 订单数量: 20
4. 点击"开始生成"
5. 等待完成

### Step 6: 开始使用！

✅ 现在你可以：
- 浏览商品
- 发布商品
- 购买商品
- 查看订单
- 查看数据看板

## 🎯 常见启动问题

### ❌ 问题1: ModuleNotFoundError: No module named 'ttkbootstrap'

**解决方案：**
```bash
pip install ttkbootstrap
```

### ❌ 问题2: 客户端无法连接服务器

**解决方案：**
1. 确认服务器已启动
2. 检查防火墙设置
3. 确认端口8888未被占用

### ❌ 问题3: 端口被占用

**解决方案：**
修改 `common/config.py` 中的端口号：
```python
SERVER_PORT = 9999  # 改为其他端口
```

### ❌ 问题4: 图表中文乱码

**解决方案：**

**Windows:** 自带黑体，无需处理

**Linux:**
```bash
sudo apt-get install fonts-noto-cjk
```

**Mac:**
```bash
# 安装中文字体
brew install font-source-han-sans
```

## 💡 使用技巧
